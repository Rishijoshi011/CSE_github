6. What is missing data? Explain different methods to handle missing data. 

# **Missing Data and Methods to Handle It**  

## **1. What is Missing Data?**  
Missing data occurs when values are absent from a dataset, leading to incomplete information. This can reduce the accuracy of analysis and machine learning models.  

ðŸ“Œ **Causes of Missing Data:**  
âœ” **Human Error** â€“ Mistakes in data entry, skipped survey questions.  
âœ” **Data Corruption** â€“ Loss of information due to system failures.  
âœ” **Equipment Issues** â€“ Sensor malfunctions in IoT devices.  
âœ” **Privacy Concerns** â€“ Users choosing not to disclose personal details.

## **2. Types of Missing Data**  

ðŸ”¹ **MCAR (Missing Completely at Random):**  
âœ… Missing data has no pattern and is purely random.  
ðŸ“Œ *Example:* A survey respondent accidentally skips a question.  

ðŸ”¹ **MAR (Missing at Random):**  
âœ… Missing data is related to **some observed variables** but not the missing value itself.  
ðŸ“Œ *Example:* People with high incomes **choose not to disclose salary** in a survey.  

ðŸ”¹ **MNAR (Missing Not at Random):**  
âœ… The missing data is related to **the value itself**.  
ðŸ“Œ *Example:* People with **poor health** avoiding answering health-related questions.  

---

## **3. Methods to Handle Missing Data**  

### **1. Deletion Methods**  
ðŸ“Œ **a) Listwise Deletion (Complete Case Analysis)**  
âœ” Removes entire rows with missing values.  
âœ” Useful when missing data is small (<5%).  
âŒ Can lead to data loss if many rows are removed.  
ðŸ“Œ *Example:* A dataset of 1000 students removes 10 rows where test scores are missing.  

ðŸ“Œ **b) Pairwise Deletion**  
âœ” Uses available data without removing entire rows.  
âœ” Better than listwise deletion but may cause inconsistencies.  
ðŸ“Œ *Example:* Calculating the average income using only the rows where income data is available.  

---

### **2. Imputation Methods (Replacing Missing Values)**  
ðŸ“Œ **a) Mean/Median/Mode Imputation**  
âœ” Replaces missing values with the mean, median, or mode of the column.  
âœ” Works well for numerical data but may introduce bias.  
ðŸ“Œ *Example:* Missing ages in a dataset are filled with the **average age** of available records.  

ðŸ“Œ **b) K-Nearest Neighbors (KNN) Imputation**  
âœ” Fills missing values using **similar data points** from K-nearest neighbors.  
âœ” Works well for datasets with relationships between variables.  
ðŸ“Œ *Example:* If a studentâ€™s test score is missing, it is estimated using scores from **similar students**.  

ðŸ“Œ **c) Regression Imputation**  
âœ” Uses a regression model to predict missing values based on other features.  
âœ” More accurate but complex.  
ðŸ“Œ *Example:* Missing salary data is predicted using **age, education, and experience.**  

ðŸ“Œ **d) Forward/Backward Fill (For Time-Series Data)**  
âœ” **Forward Fill:** Fills missing values with the **previous** available value.  
âœ” **Backward Fill:** Fills missing values with the **next** available value.  
ðŸ“Œ *Example:* Missing temperature data for 3 PM is filled using **2 PMâ€™s temperature.**  

---

### **3. Advanced Methods**  
ðŸ“Œ **a) Multiple Imputation**  
âœ” Generates multiple estimates for missing values and averages them.  
âœ” More accurate but computationally expensive.  

ðŸ“Œ **b) Deep Learning-Based Imputation**  
âœ” Uses AI models like **Autoencoders** to predict missing values.  
âœ” Best for large and complex datasets.
